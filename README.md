# magenta-web

![Magenta Orb Visualization](./docs/assets/magenta-orb-preview.gif)

Веб-проект с процедурной анимацией orb на `Vite + React + React Three Fiber + Three.js`.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Что внутри

- `React`
- `TypeScript`
- `three`
- `@react-three/fiber`
- `@react-three/postprocessing`
